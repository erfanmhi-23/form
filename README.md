# Form

سامانه ایجاد و مدیریت فرم‌های پویا (داینامیک)

## توضیحات
هدف این پروژه، ایجاد یک سامانه با قابلیت ذخیره‌سازی، نمایش و نظارت بر فرم‌ها و فرآیندهای پویا و گزارش‌گیری از نتایج آنهاست.  
با توسعه این سامانه، قصد داریم سیستمی پیاده‌سازی کنیم که کاربران از طریق آن بتوانند فرآیندهای دلخواه خود را ایجاد و دسترسی افراد مختلف به آن را مدیریت کنند.

آدرس پیش‌فرض اجرای پروژه: [http://localhost:8080](http://localhost:8080)

## امکانات اصلی
- ایجاد و مدیریت فرم‌های پویا
- اشتراک‌گذاری لینک منحصر به فرد فرم‌ها
- مشاهده نتایج و گزارش‌گیری
- API کامل با مستندات (`api.doc` موجود)

## پیش‌نیازها
- Docker و Docker Compose
- Python >= 3.11
- Redis
- PostgreSQL
- graphql
- celery
- nginx
